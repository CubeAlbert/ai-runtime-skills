# URL Shortener Service

Build a URL shortener web application. The interviewer wants to see a working demo.

## Core Requirements

- User registration and login (email + password)
- Authenticated users can create short URLs from long URLs
- Visiting a short URL redirects to the original long URL
- Click tracking: record each visit with timestamp and referrer
- Analytics dashboard showing click counts per URL and over time

## Tech Stack

Not specified — choose whatever is appropriate. The interviewer mentioned they're comfortable reviewing any stack.

## Requirements for Demo

- The interviewer wants to see a working demo, so it needs a web UI
- Include automated tests for the core redirect and URL creation logic
