# 批量文件重命名工具 (Batch File Renamer)

开发一个命令行工具，用于批量重命名文件。

## 核心功能

- 支持正则表达式匹配和替换文件名
- 在执行重命名之前展示预览（dry-run 模式），让用户确认
- 支持撤销上一次重命名操作
- 支持递归处理子目录
- 支持大小写转换（大写转小写、小写转大写、首字母大写）

## 技术要求

- 使用 Go 语言实现
- 跨平台支持（Windows, macOS, Linux）
- 编译为单个可执行文件，不依赖运行时

## 加分项

- 支持从文件读取重命名规则（JSON 配置文件）
- 输出彩色终端提示
- 包含完整的单元测试
